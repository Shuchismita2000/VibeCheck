# WanderLore: AI Destination & Cultural Discovery Platform

This plan outlines replacing the current ChefFlow codebase with a new application tailored exactly to the "Destination Discovery & Cultural Experiences" hackathon challenge.

## User Review Required

> [!WARNING]
> This plan will completely overwrite the existing ChefFlow codebase (index.html, src/main.js, src/style.css, etc.) with the new WanderLore travel application. Please confirm you are okay with deleting the previous project files to start fresh for this challenge.

## Goal Description

Build a GenAI-powered web platform ("WanderLore") that helps travelers discover destinations and engage with local culture. To meet the hackathon requirements and ensure you are in the top 10, the app will:
1. Cover all perspectives: recommend attractions, uncover hidden gems, generate immersive storytelling, promote heritage, suggest local events, and connect with authentic experiences.
2. Be highly aesthetic: premium glassmorphic UI, modern typography, and smooth micro-animations.
3. Be fully functional end-to-end: using real Gemini API calls (no hardcoded data or hallucinations), allowing the evaluator to input their API key.

## Proposed Changes

We will restructure the frontend using Vanilla JS, HTML, and CSS to keep it lightweight but visually stunning.

### Core Application Setup

#### [MODIFY] index.html
- Overwrite with a new responsive layout.
- Add an API Key configuration modal for evaluators (crucial for "no fake data" rule).
- Main Dashboard: 
  - Search bar for destination and travel dates.
  - **New Input**: "Your Origin Country" (to dynamically adjust how Gemini writes the story - introductory for foreigners, deeper/nuanced for locals).
- Output Sections:
  - **Heritage Storyteller**: An immersive story about the location's history generated by AI. Will feature a stunning, dynamically fetched image (e.g., using Pollinations.ai or Wikipedia API).
  - **Hidden Gems & Attractions**: AI-recommended spots off the beaten path, complete with AI-fetched images for each location.
  - **Local Culture & Events**: Suggestions for authentic food, crafts, and current events.

#### [MODIFY] src/style.css
- Implement a stunning, dark-mode focused, glassmorphic design system.
- Curated color palette (e.g., Deep oceanic blues, vibrant sunset oranges as accents).
- Micro-animations for card hover states, loaders (e.g., a glowing compass), and smooth transitions.

#### [MODIFY] src/main.js
- Handle UI interactions, modal states, and input gathering.
- Orchestrate the generation of the cultural guide when a user searches.

#### [MODIFY] src/gemini.js
- Implement the actual prompt engineering to hit the Gemini API.
- The prompt will compare the user's origin country with the destination country. If they match, the story will be nuanced and tailored to a local. If they differ, the story will be an engaging introductory perspective.
- Ensure the prompt strictly requests structured JSON responses covering:
  - `storytelling`: A rich paragraph about the destination's heritage.
  - `storytellingImagePrompt`: A detailed visual prompt to fetch an image for the story.
  - `attractions`: Array of top spots, why they matter culturally, and a specific image search keyword/prompt.
  - `hiddenGems`: Array of lesser-known authentic locations, and an image search keyword/prompt.
  - `localEvents`: General types of local events/festivals the traveler should look for.
  - `culturalExperiences`: Specific foods, crafts, or local traditions to engage in.
